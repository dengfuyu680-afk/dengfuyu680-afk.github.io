Used for overall application analysis, including error resolution, troubleshooting, and debugging; improving services through general analytics; reviewing performance and payment processes; ensuring smooth game operation; and addressing technical issues like bug fixing.

Includes name, Open ID, device ID, application version, battery status, Wi-Fi signal strength, phone storage, network type, iOS version, platform, carrier, country code, and any uploaded photos to illustrate issues. Includes device model, name, manufacturer, running process data, SDK version, and application package name. Collected to send you push notifications about announcements, updates, service-related information, application versions, carrier details, and network types.
